# 🌼 Marigold Café

A responsive, multi-page concept website for a modern café brand — built to demonstrate clean UI, component-based architecture, and smooth client-side routing.

**Live Site →** [marigoldcafe.netlify.app](https://marigoldcafe.netlify.app) *(update with actual URL)*

-----

## Overview

Marigold Café is a frontend concept project simulating a real-world café business site. It showcases my ability to build polished, production-ready interfaces using modern React tooling — from component structure to deployment.

-----

## Tech Stack

|Tool        |Purpose               |
|------------|----------------------|
|React       |UI component library  |
|Tailwind CSS|Utility-first styling |
|React Router|Client-side navigation|
|Netlify     |Hosting & deployment  |

-----

## Features

- Multi-page layout with React Router (no full-page reloads)
- Fully responsive across mobile, tablet, and desktop
- Clean, consistent design system using Tailwind utility classes
- Deployed and live via Netlify

## Getting Started

```bash
# Clone the repo
git clone https://github.com/Samson-Oruekpeti/Mari-G.git
cd Mari-G

# Install dependencies
npm install

# Run locally
npm run dev
```

Open <http://localhost:5173> in your browser.

-----

## Project Structure

```
src/
├── components/     # Reusable UI components (Navbar, Footer, etc.)
├── pages/          # Route-level page components
├── assets/         # Images and static files
└── App.jsx         # Root component with route definitions
```

-----

## Deployment

This project is deployed on **Netlify** with automatic builds from the main branch.

To deploy your own copy:

1. Push the repo to GitHub
1. Import it on [netlify.com](https://netlify.com)
1. Set build command: `npm run build`
1. Set publish directory: `dist`
